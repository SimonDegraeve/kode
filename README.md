## Install

```
$ npm install --global kode
```


## Usage

```
$ kode --help

  Usage
    $ kode <command> <options>

    Command can be:
      lint | test | clean | transpile | build | release | report-coverage | changelog

  Examples
    $ kode lint
    $ kode test --watch
```
